# 简化简单生图工作台

## 目标

把 `apps/web` 的“简单生图”页面从表单式配置页改成低干扰的生图工作台，参考 `/Users/jesuscc/wcc/projects/github/chatgpt2api/web/src/app/image` 的工作台结构，让用户先输入提示词并快速生成，再在需要时调整参数或进入画布继续编辑。

## 用户价值

- 新用户第一眼能理解：这是输入描述并生成图片的地方。
- 常用用户能更快生成：默认值可直接用，尺寸、张数、风格不再占据主视觉。
- 专业能力保留：积分、公开状态、高级尺寸、质量、格式、生成结果、进入画布仍可访问，但默认视觉更轻。
- 当前产品的本地优先、画布延续、图片广场公开控制不丢失。

## 已确认事实

- 当前页面入口为 `/generate`，实现位于 `apps/web/src/features/simple-generation/SimpleGenerationPage.tsx`，样式位于 `apps/web/src/styles/simple-generation.css`。
- 当前页面将提示词、尺寸卡片、全量尺寸下拉、风格、数量、公开开关、高级设置、积分、状态、结果面板全部平铺在首屏，信息量偏高。
- 参考项目工作台位于 `/Users/jesuscc/wcc/projects/github/chatgpt2api/web/src/app/image/page.tsx`，核心组件为 `components/image-composer.tsx`、`components/image-results.tsx`、`components/image-sidebar.tsx`。
- 参考工作台的有效模式是：左侧历史记录，中间结果/空状态，底部大输入框，参数压缩成底部胶囊控件，生成按钮是右侧圆形图标。
- 项目规范要求保持 warm paper、ink、copper、teal 视觉系统，不能直接复制参考项目的 Tailwind 石色主题，也不能做营销式落地页。
- `apps/web` 用户可见文案需要同步写入 `apps/web/src/shared/i18n/index.tsx` 的中文和英文。
- UI 完成前需要运行 `pnpm typecheck`、`pnpm build`，并用浏览器检查桌面和移动视口。

## 需求

- 首屏改为“工作台”布局：内容区居中，底部固定/贴底的输入工作台成为主操作入口。
- 移除当前首屏大表单感：尺寸卡片、风格下拉、数量按钮、公开开关、积分摘要、高级设置不再全部平铺。
- 保留当前生成 API 行为：提示词、尺寸、风格、质量、格式、数量、公开状态、积分校验、生成中/失败/成功反馈、结果进入画布。
- 结果区改成对话/工作流感：没有结果时显示轻量空状态；有结果时显示最近生成内容，可复制提示词、下载、发送到画布、查看更多。
- 参数入口改成轻量控件：默认只展示张数、比例/尺寸、风格或参数入口，其他设置进入弹层/抽屉/菜单。
- 页面应适配移动端：移动端历史/结果不横向溢出，底部输入区按钮和文字不重叠。
- 页面文案需要减少解释感，首屏避免长段说明。

## 范围决定

- 本任务纳入上传/粘贴参考图能力。无参考图时调用 `/api/images/generate`，有参考图时调用 `/api/images/edit`。
- 参考图只作为本次输入工作台状态，不引入新的长期会话存储。

## 暂定不做

- 不改顶部导航顺序。
- 不改 API、积分规则、Gallery 数据结构或公开作品规则。
- 不重做画布页或 Agent 流程。
- 不直接复制 `chatgpt2api` 的样式系统、依赖或状态存储。

## 验收标准

- [x] `/generate` 首屏视觉变为生图工作台：主输入区优先，参数不再像长表单一样铺满首屏。
- [x] 用户不填写提示词时，页面能通过空状态/预设引导创作，不出现大面积空白结果面板压迫主操作。
- [x] 用户填写提示词后，可以用默认参数直接生成图片。
- [x] 张数、比例/尺寸、风格/参数、公开状态、积分消耗仍可查看或调整。
- [x] 用户可以上传或粘贴参考图；存在参考图时生成走编辑模式，且可以移除参考图。
- [x] 生成中、成功、失败、积分不足、服务不可用状态仍有清晰反馈。
- [x] 最近结果仍支持下载、复制提示词、发送到画布、查看更多。
- [x] 中文和英文文案同步。
- [x] 桌面和移动视口无明显文字溢出、控件重叠、底部输入遮挡关键内容。
- [x] `pnpm typecheck` 通过。
- [x] `pnpm build` 通过。
- [x] 浏览器验证通过：复用本地 `http://localhost:5173/generate` 检查桌面和移动视口。
