# 替换简单生图固定预设

## 目标

把 `/generate` 新创作空状态里的 4 个预设替换为 `https://image.deepark.tech/image` 首页展示的 4 个高质量生图预设，包括图片、比例、标题、说明和套用后的完整提示词。

## 背景

当前本地简单生图的预设来自提示池随机内容，风格和文案质量不稳定。用户明确要求参考已打开的 `image.deepark.tech/image` 页面，把页面上 4 个预设固定保存下来，替换本地当前的套用预设。

## 需求

- 使用 Chrome / Browser Bridge 抽取远端页面 4 个预设：
  - 轮廓宇宙海报
  - 青花瓷博物馆图鉴
  - 古风联动宣传图
  - 地平线深圳实机图
- 保存每个预设的封面图片到本地可提交的静态资源目录。
- 点击远端“套用这个预设”，获取每个预设填入输入框后的完整提示词。
- 本地 `/generate` 新创作空状态固定展示这 4 个预设，不再从提示池随机取。
- 本地卡片保留“套用这个预设”操作；点击只填充输入框，不触发生成。
- 保持当前暖色背景、底部工作台、左侧最近作品逻辑。

## 不做

- 不调用生成接口。
- 不消耗额度。
- 不改提示池、Gallery、生成记录、画布数据结构。
- 不引入新依赖。

## 验收标准

- [x] 本地 `/generate` 默认新创作展示 4 个固定预设。
- [x] 4 个预设的图片、标题、说明、比例与远端页面一致。
- [x] 点击每个预设填入对应完整提示词。
- [x] 本地不再展示提示池随机预设。
- [x] 桌面和移动端无横向溢出，卡片不重叠。
- [x] `pnpm typecheck` 通过。
- [x] `pnpm build` 通过。
- [x] 内置 Browser 验证 `http://127.0.0.1:8787/generate` 通过。

## 实现记录

- 通过 Chrome / Browser Bridge 抽取了远端首页 4 个预设卡片的图片 URL、标题、说明、比例和套用后的完整提示词。
- 将封面图片保存到 `apps/web/public/simple-presets/`。
- 将 `/generate` 空状态预设改为固定引用这 4 个模板，不再读取提示池或最近作品作为预设来源。
